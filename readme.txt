== Installation ==
1. Unzip supaz-instagram-feeds-lite.zip
2. Upload all the files to the /wp-content/plugins/supaz-instagram-feeds-lite
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For adding the feeds, click on Supaz Instagram Feeds lite menu in WordPress left admin menu.
5. Please configure the plugin by going to Instagram settings. Configure the access token and option to enable links on image caption.
6. To create new shortcode click on add new menu and configure the settings. 
7. After configuring the feed copy the shortcode and use it on posts and pages where needed.
